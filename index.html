What is GetSharing.io?

It's a webapp that helps any group of people, manage a communal pile of physical objects called shareables.

Anyone can easily add a shareable to the communal pile. Once it's there, any user is able to borrow it and commit to a return date. All users can see who's got what shareable and when it's due back in the pile.

Sharing is simple

  • Who are you? Identify yourself, so we all know who's doing the borrowing.
  • What would you like to borrow? Identify the shareable that you are borrowing from the communal pile.
  • When will you return it? Let the others know when they can have a go. We'll remind you the day before.
